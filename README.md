# Penguime 🐧  
*A simple 2D Java Swing penguin adventure*

## Controls
- **Movement**: WASD  
- **Interact**: P (pick up), SPACE (use)  
- **Inventory**: 1-9 (select), 0 (deselect)  
- **System**: ESC (pause), F4/F5 (save/load)  

## Run
1. Install [JDK 17+](https://adoptium.net/)
2. Build & play:
```bash
git clone https://github.com/AbdullahMakhos/Java-Game.git
cd Java-Game
mvn package && java -jar target/myGame-0.0.1-SNAPSHOT.jar
